Chat Archive - 前端媒体渲染测试

测试 1: 单个表情包 - 模拟用户问题场景

场景：对方只发送一个 GIF 表情包
预期：只显示一个媒体块，无"图片不可用"，无临时路径

测试 2: 图片消息 - DB media 存在

场景：DB 已归档图片
预期：显示一个图片媒体块（compact 模式）

测试 3: 混合消息 - 文字 + 图片

场景：文字和图片混合
预期：文字正常显示，图片只显示一次

测试 4: 占位文本过滤

场景：raw 里只有 [图片] 占位文本，DB media 存在
预期：占位文本被过滤，只显示 DB media

测试 5: 多个 raw 候选源

场景：raw 里有多个图片元素，DB media 只有一个
预期：只显示 DB media，raw 元素被跳过
